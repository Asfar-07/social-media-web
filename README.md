> A full-featured social media platform built with Node.js, Express, and Socket.IO — my first college team project (2023–2024).

This project holds a special place for me. After losing it for some time, I was able to recover it,
and every line of code brings back meaningful memories from my learning journey.

Built as a **team effort**, where we faced real challenges — security concerns, code readability,
and designing a functional platform from scratch while learning real-world development practices.

---

## 🚀 Features

- 🔐 Secure user authentication system
- 👤 User-friendly account dashboard
- 💬 Public and private chat (real-time with Socket.IO)
- 📝 Examination module
- 🎮 Gaming section
- 🔍 Search functionality

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| Node.js | Backend runtime |
| Express.js | Web framework |
| Handlebars | Templating engine |
| Socket.IO | Real-time communication |
| CSS | Styling |
| MongoDB | Database |

---

## ⚙️ Setup Instructions
```bash
# 1. Clone the repository
git clone https://github.com/Asfar-07/social-media-web.git

# 2. Navigate into the project
cd social-media-web

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev

# 5. Open in browser
http://localhost:8000
```

## 📁 Project Structure
```
social-media-web/
├── bin/             # Server entry point
├── connection/      # Database connection
├── databaseSchema/  # MongoDB models
├── helper/          # Helper functions
├── public/          # Static files (CSS, JS, images)
├── routes/          # Express routes
├── views/           # Handlebars templates
├── uploads/         # User uploaded files
└── app.js           # Main app file
```
